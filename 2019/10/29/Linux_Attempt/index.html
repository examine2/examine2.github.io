Linux Attempt

虚拟机备份

将准备好的disk1和disk2等文件通通复制到一处，原Linux损坏后直接复制回去覆盖，在Vaware打开虚拟机

目录

在Linux中，一切都是文件

/ （根目录）

  • home 存放用户当前信息
  • bin 可执行的命令
  • boot 系统引导程序
  • etc 系统配置
  • dev 设备驱动

账户

普通账户： 用户自行创建的账户
超级账户： 默认名root，拥有控制普通账户读写、关机的权力

命令

su (substitute)

切换普通账户和超级账户

$表明当前处于超级账户下，#则表明处于普通账户

shutdown -h now

root账户特权，马上关机

reboot

重启

cd (change directory)

切换到指定目录，可使用相对路径或绝对路径
Tips:

  • 可以使用Tab键补全
  • cd ..退回到上一级目录

pwd (print working directory)

打印当前工作目录

ls (list)

列出当前目录包含的所有目录

mkdir (make directory)

创建一级文件夹
可以创建多级文件夹，比如，创建三级文件夹：mkdir dir1/dir2/dir3；但当dir1和dir2不存在时，该命令报错，可设置-p参数直接创建：mkdir -p dir1/dir2/dir3

touch

创建普通的文本文件，相当于Windows下的.txt
（注意，.txt和.exe等Windows常见文件在Linux系统中都是无效的文件后缀，将被视为普通文本文件）

man (manual)

manual [ˈmænjuəl] n.说明书 adj.手工的
查看指定命令的详细信息，如man mkdir；进入手册后，按q退出

clear

执行清屏操作

file

用法：file 'filename'
判断文件夹/文本文件，返回显示directory/empty/text

rm (remove)

删除普通文本文件(有提示)，传递-rf参数强制删除

-f (force) 强制删除，rm默认只删除普通文本文件，不删除文件夹
-r/-R (recursive) 递归删除，当目录下含多级目录，rm将往下访问，逐级删除

top

动态查看任务管理器

ps -au (processes snapshot)

snapshot [ˈsnæpʃɒt] v.拍快照 n.简介
静态查看任务管理器，得到一张瞬间快照

-au (all user)

histroy

查看最近执行的所有指令(默认1000条)
罗列出来的指令拥有序号，重复执行指令只需!+序号

find

用法：find 'dirname' -name 'filename'
根据文件名进行搜索，返回路径；支持模糊搜索，用*代替

grep (global regular expression print)

用法：grep [option] pattern *file
应用正则表达式，打印符合模式的列
常用的作用是与find指令通过|形成管道，接收find指令传递过来的数据，进行进一步的筛选

tar (tape archive)

archive [ˈɑːkaɪv] v.存档，归档 n.档案，档案馆
磁带归档，进行压缩文件有关的操作
Linux中的压缩文件大多以.tar.gz为文件后缀名

压缩文件 tar -czvf (create gzip verbosely file)

用法：tar -czvf 压缩后文件 压缩前文件
参数：

-c (create) 创建压缩文件
-z (gzip) 使用压缩算法gzip
-v (verbosely) 显示创建文件的全过程
-f (file) 文件

命令的参数有一些是允许随意调换位置的，如tar -cvzf等效于tar -czvf；但这里的-f后面紧跟文件名，所以必须放在最后

解压文件 tar -zxvf (gzip extract verbosely file)

用法：tar -zxvf 文件名
参数：

-x (extract) 撤销压缩，即解压

可以在解压时添加-C参数，指定解压的位置：tar -zxvf 文件名 -C 解压位置

ifconfig (interface configuring)

查看Linux的ip地址，该命令替代了Windows下的ipconfig(internet protocol configuration)命令

service

与防火墙有关

service iptables stop 清除防火墙
service iptables start 启用防火墙

rpm (Red-Hat package manager)

包管理器，用于生成、安装、查询、更新、卸载某个软件包
查看已安装的包：rpm -qa | grep [pattern
参数：

-q (query) 访问，后接名称，查询指定的包是否被安装
-a (all) 指定范围为所有
卸载已安装的包：rpm -e --nodeps <package-name>
参数：
-e (delete) 卸载包
–nodeps (no dependence) 卸载前不检查依赖性，完全卸载

vi (vim)

vim是所有Unix Like系统都会内建的文本编辑器，类似Windows下的记事本
指令vi 'filename'用vim打开指定的文本文件

cat (concatenate)

直接浏览文件内容，本质是连接文件并在标准输出上输出

more

在显示器上阅读文件的过滤器，分页显示

less

按页或窗口打印文件内容
（less是more的增强，按需显示文件内容，在查看大文件时非常有用）
快捷键：

ctrl+b 往上翻页
ctrl+f 往下翻页
g 移动到第一行
G 移动到最后一行

kill

结束进程
用法：kill -singal pid
参数：

-pid 进程编号，可以使用top或ps指令查看
-singal 信号，辅助作用

kill pid可能出现kill不成功的现象，最常用的信号就是-9，进行强制杀死进程
可以通过kill -l命令来查阅更多有关kill的信号参数

ping

检查当前网络与目标网络是否连通，输入待连通网络的ip
测试通过发送包，显示丢包率，可按ctrl+C强制结束
输出格式为：
xx packets transmitted, xx received, xx packet loss, time xx

虚拟机Linux下安装软件(以JDK为例)

方法① yum

通过指令yum在线安装

方法② WinSCP

连通

（校园网使用客户端Dr.COM连接网络，但Linux还没有Dr.COM，无法连接上网络，因此建议用手机热点网络）
在Windows的cmd窗口中输入ipconfig获取ip，再在Linux终端中输入ifconfig获得ip

传送

将提前下载好的tar.gz包置于Windows中，然后打开WinSCP软件，输入Linux系统的ip号，默认端口为22，使用root账户
随后在打开的双窗口中找到tar.gz包，将其拖曳到Linux系统中即可

解压

直接使用命令tar -zxvf jdk...进行解压
